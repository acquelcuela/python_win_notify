投稿用画像生成ツールの候補(2026年7月時点)
====================================

Nano Banana(Gemini)
--------------------
- 日本語のテキスト描画が得意
- 無料枠あり
- このプロジェクトで既に使っているGEMINI_API_KEYと同じ仕組みで使えるため、
  将来的に画像生成まで自動化したくなった時に相性が良い
- おすすめ度: ◎(まず試すならこれ)

Bing Image Creator(Microsoft / DALL-E 3 / MAI-Image-1)
--------------------------------------------------------
- 完全無料、Microsoftアカウントでログインするだけ
- 1日15回まで高速生成
- グラフや数字入りの画像だとテキストが崩れやすい傾向

Ideogram
--------
- 画像内の文字(見出し・数字)をきれいに描画するのに定評あり
- 投資系の「タイトル＋数字を大きく見せるカード」的な画像に向いている
- 無料枠は生成回数少なめ

Canva(Magic Media + テンプレート)
----------------------------------
- 純粋なAI画像生成というより、AI生成した背景に自分で数字やグラフを
  正確に配置できるのが強み
- 高配当株の比較表・利回りシミュレーションのような「数字が正確じゃないと
  意味がない」系の画像なら、AI単体よりCanvaの方が実用的なことが多い

使い分けの目安
==============
- 雰囲気重視の1枚絵 → Nano Banana か Ideogram
- 数字を正確に見せたい図解 → Canva

次のステップ
============
画像が準備できたら、自動ピックアップ→投稿の仕組み作りに着手する。

画像生成プロンプト案(文字なし・投資っぽさ・目を引く)
====================================================

前提
----
- Nano Banana(Gemini)は日本語プロンプトでも英語と同レベルで理解する。
  他の一部の画像生成ツール(純粋な拡散モデル系)は英語の方がやや精度が
  出る傾向があるが、Nano Banana中心なら日本語でよい。
- 「文字・数字・ロゴは入れない」の一文は毎回必ず入れておくと、
  崩れ字が出る確率がかなり下がる。
- スタイル指定の単語(フラットイラスト・アイソメトリック・ミニマル等)は
  そのままカタカナで書いても伝わりやすい。

方向性A: かわいい動物×お金の成長(おすすめ)
--------------------------------------------
動物ものはX(旧Twitter)で停止率(スクロールを止める力)が強く、
投資系の硬い話題と組み合わせると差別化になる。

1.
まんまるとした柴犬が、金貨の葉っぱがついた小さなお金の木の横に座っている
イラスト。朝の柔らかい光、パステル調の暖色、フラットイラストスタイル、
背景はシンプルでミニマル。文字・数字・ロゴは入れない。

2.
小さなカピバラが、葉っぱの代わりに金貨がなる植物に水をやっているイラスト。
温かみのある雰囲気、緑と金を基調にした配色、アイソメトリックスタイル、
背景はシンプル。文字・数字・単語は入れない。

3.
小さくて丸い鳥が、階段のように積み上がった金貨の山の上に座っている
イラスト。朝の光、暖色系の配色、フラットなベクターイラスト。
文字・数字は入れない。

方向性B: 抽象・ミニマルな「成長」モチーフ
------------------------------------------
数字やグラフを直接描かせると崩れやすいので、階段状の積み上げや
右肩上がりの造形を「もの」として表現させると安全。

4.
金貨が階段状に右肩上がりに積み上がっている、ミニマルなフラット
イラスト。背景は緑と金の柔らかいグラデーション、アイソメトリック
スタイル、シンプルで清潔感がある構図。文字・数字・グラフの線は
入れない。

5.
窓辺に置かれた鉢植えで、葉っぱが金貨の形をしている植物のイラスト。
朝の日差し、暖色系のミニマルな配色、フラットデザイン。文字・単語は
入れない。

方向性C: 世界観のある1枚絵
----------------------------
6.
巨大な金貨の上に小さな人物が立ち、街並みに昇る朝日を眺めている
イラスト。希望を感じる雰囲気、フラットイラストスタイル、金と紺の
配色。文字・数字・ロゴは入れない。

運用のコツ
----------
- 色を統一する(例: ゴールド×グリーン、またはゴールド×ネイビー)と、
  タイムラインで「あのアカウントだ」と認識されやすくなる。
- 同じ動物や同じ構図を"シリーズ化"すると、プロフィール全体としての
  世界観が作れる(例: いつも同じ柴犬キャラが登場する、など)。
